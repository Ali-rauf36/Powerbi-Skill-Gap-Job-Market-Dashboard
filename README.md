# Skill Gap VS Job Market Dashboard — Power BI

## Overview
A dark theme interactive dashboard built in Power BI analyzing 
skill demand vs candidate hiring across 500 records and 6 
skill categories.

## Features
- Dark theme canvas with custom color palette
- 6 DAX measures including Hiring Rate %, Avg Salary
- Grouped bar chart showing demand by skill category
- Dynamic slicers — Industry · Year · Location · Experience
- Donut charts for hiring status and skill distribution
- Column chart for job demand by industry

## Tech Stack
- Power BI Desktop
- DAX
- Power Query (M language)

## Dataset
- Records: 500 candidates
- Skills: Management · Finance · Programming · Marketing · Analytics · Communication
- Markets: Education · IT · Manufacturing · Healthcare · Retail · Finance
